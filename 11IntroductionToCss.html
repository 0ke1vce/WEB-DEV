About Css

  • Css give style to raw html
  • give style to our web page
  • use to make website responsive
  • Cascading style sheet (CSS) take responsibility of design.

CSS Syntax

p{color:blue;} selector{property:value;}

For class

.className{property:value;}

For id

#idName{property:value;}

For group of selectors

selector1,selector2{property:value;property2:value2;}

  1. Inline Css: added to element directly using style attribute.
  2. Internal Css: Css is kept inside head tags in <style> tags.
  3. External Css:Css is kept seperately inside a .css style sheet.
